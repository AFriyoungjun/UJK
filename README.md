## 📖 Deskripsi Proyek
Proyek ini merupakan sistem manajemen booking lapangan berbasis web yang dirancang untuk memudahkan pengguna dalam melakukan pemesanan jadwal bermain secara online. Sistem ini menampilkan ketersediaan jadwal secara real-time dan memungkinkan pengguna untuk memilih waktu yang tersedia dengan mudah.

Selain itu, sistem juga dilengkapi dengan dashboard admin untuk mengelola data booking, sehingga proses pengelolaan jadwal menjadi lebih terstruktur, efisien, dan transparan.

## 🚀 Fitur Utama

### 👥 User (Pengguna)
- Melihat jadwal ketersediaan lapangan
- Memilih waktu booking secara real-time
- Melakukan pemesanan melalui form booking
- Mendapatkan informasi status jadwal (kosong/terisi)

### 📝 Form Booking
- Input nama pemesan
- Pilih jenis lapangan
- Tentukan jam mulai
- Pilih durasi bermain
- Informasi metode pembayaran

### 🛠️ Admin
- Melihat seluruh data booking
- Mengelola (edit & hapus) data pemesanan
- Monitoring jadwal lapangan
- Update status booking

### 📅 Sistem Jadwal
- Tampilan jadwal berbentuk grid waktu
- Indikator slot kosong dan terisi
- Informasi nama pemesan pada jadwal
- Update otomatis setelah booking
